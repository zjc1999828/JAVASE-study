# 接口的多态特性
同继承的多态特性

向上转型
接口类型的引用可以指向实现了接口的类的对象实例。


向下转型
同继承。


向上转型和向下转型两种方法中依然遵循动态绑定机制和继承机制。



##### 接口的多态传递机制
接口A继承了接口B
则当类C实现接口A的时候，类C也就实现了接口B。

很好理解啊。儿子把父亲的所有东西都学会了。别人实现了儿子，自然而然也就实现了父亲。
