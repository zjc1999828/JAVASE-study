常见用法：

```bash
public class main1{
    public static void main(String[] args) {
        outer out = new outer();
        out.f2();
    }
}

class outer {
    private int ag1 = 3;
    public void  f1(){
        System.out.println("outer中的f1()被call");
    }
    public void f2(){
        System.out.println("???");
        final class inner{
            private int ag2 = 32;
            public void f3(){
                System.out.println("inner的f3()被call");
            }
            public void f4(){
                System.out.println(outer.this.ag1);
            }
            public void f5(){
                f2();
            }
        }
        inner in = new inner();
        in.f4();
    }
}
```
