注意delete只能删除空目录，即当目录中有其他文件的时候，是不能够进行删除delete操作的！




madir创建单级目录

```java
    @Test
    public void create(){
        String path = "D:\\sksk";
        //下面这句话就是在娘胎里孕育了一个对象的意思。但是并没有在硬盘里面把它生出来.
        File ff = new File(path);
        ff.mkdir();
    }
}
```

madirs创建多级目录

```java
   @Test
    public void create(){
        String path = "D:\\demo\\a\\b\\c";
        //下面这句话就是在娘胎里孕育了一个对象的意思。但是并没有在硬盘里面把它生出来.
        File ff = new File(path);
        ff.mkdirs();
    }
}

```
