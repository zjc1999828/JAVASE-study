# 三元运算符的练习——实现三个数中的最大值

```java
public class c8{
	public static void main(String[] args){
		int a = 3002;
		int b = 3002;
		int c = 213;
		int max = 0;
		max = a >= b ? a : b;
		max = max >= c ? max : c;
		System.out.println(max);
	}
}
```
- 若用一句话实现

```java
public class c8{
	public static void main(String[] args){
		int a = 3002;
		int b = 3002;
		int c = 213;
		int max = 0;
		max = (a >= b ? a : b) >= c ? (a >= b ? a : b) : c;
		System.out.println(max);
	}
}
```
